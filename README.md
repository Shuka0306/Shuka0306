## 最近在做

- 研究 **gVisor / sandbox** 场景下的异步 I/O，尤其是它的边界和退化条件。
- 整理 **论文实验** 和 **benchmark harness**，把每个结论都落到可复现的数据上。
- 读 **容器运行时**、**文件系统**、**存储系统** 的实现，顺手理解它们为什么会那样工作。
- 把实验结果整理成能复现、能解释、也能顺利写进论文的形式。
