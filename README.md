## About Me

- I am a university student in China, learning computer systems step by step.
- My main language is **Go**, and I also spend time with **C**, **TypeScript**, **Docker**, and some **C++/Python** when needed.
- I am especially interested in the part of software that gets close to the system:
  Linux internals, networking, container runtime design, Kubernetes datapath, and performance-related tooling.
- I prefer learning by building and reading real code, not only by reading notes.
